# Overthinking Calculator

Note:
This calculator was created for fun. It is not meant to replace a calculator you would use every day.

---

## Installation

1. Install .NET SDK  
https://dotnet.microsoft.com/download

2. Clone repository or download ZIP

3. Open the project folder

4. Run:

```bash
dotnet run
```
----

## Features

### Basic math
- Calculates simple expressions (e.g. 2 + 2 * 2)
- Supports basic operations (+, -, *, /)

### Trigonometry
- sin
- cos
- tan
- cot
- sec
- csc

### Extra math
- Square root (√)
- Power (x^y)
- Logarithm (ln)
- Rounding
- Absolute value (|x|)
- Random number generator 🎲
